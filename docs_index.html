CRAM Documentation

Welcome to the Cognitive Robot Abstract Machine (CRAM) documentation hub.

This monorepo contains documentation for three main projects:

Projects

krrood

Knowledge Representation and Reasoning for Object-Oriented Domains

A framework for knowledge representation and reasoning in object-oriented domains.

pycram

Python Cognitive Robot Abstract Machine

The Python implementation of the Cognitive Robot Abstract Machine, providing tools for robot task planning and execution.

semantic_digital_twin

Semantic Digital Twin

A semantic digital twin framework for robotics applications.

About CRAM

CRAM is a cognitive architecture for robot control. For more information about the overall project, installation instructions, and contribution guidelines, please visit the main repository.
